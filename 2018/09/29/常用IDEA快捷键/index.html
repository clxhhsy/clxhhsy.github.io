常用IDEA快捷键

Sep 29, 2018idea点击

Remember

动作 快捷键 说明
Show intention actions and quick-fixes Alt + Enter 智能修复
Parameter info Ctrl + P 参数提示
Shrink selection Ctrl + Shift + W 收缩选中
Search everywhere Double Shift 全局查找
Extend selection Ctrl + W 扩展选中
Generate code Alt + Insert 代码生成
Recent files popup Ctrl + E 弹出最近文件
Rename Shift + F6 重命名
Smart code completion Alt+/ 智能提示

Navigation

动作 快捷键 说明
Go to line Ctrl + G 定位到指定行号
Navigate back / forward Ctrl + Alt + Left/Right 导航向前/后退
Toggle bookmark F11 添加/删除标签（当前行）
Type hierarchy Ctrl + H 显示类结构图（类的继承层次）
Go to declaration Ctrl + B , Ctrl + Click 跳转到类/方法/变量定义处
Open quick definition lookup Ctrl + Shift + I 显示光标处类/方法/变量的定义信息
Call hierarchy Ctrl + Alt + H 显示方法被调用结构图
Go to super-method / super-class Ctrl + U 跳转到父方法/父类
Method hierarchy Ctrl + Shift + H 显示方法结构图（方法的继承层次）
Show navigation bar Alt + Home 光标定位到导航拦
Go to editor (from tool window) Esc 在工具栏窗口切换到编辑窗口
Edit source / View source F4 / Ctrl + Enter 在视图窗口中选择文件/方法等，快速跳转到编辑窗口中
Go to implementation(s) Ctrl + Alt + B 跳转到实现
Go to class Ctrl + N 查找类
Select current file or symbol in any view Alt + F1 定位文件/方法的位置在指定的视图下，如：项目结构/包目录/方法集合/磁盘位置等
Toggle bookmark with mnemonic Ctrl + F11 以指定数字或者字母做为添加书签；删除书签
File structure popup Ctrl + F12 显示当前文件的结构，可以进行方法快速过滤定位
Go to next / previous editor tab Alt + Right/Left 切换到右边/左边的编辑窗口
Next / previous highlighted error F2 / Shift + F2 定位到上一个/下一个错误或警告，并高亮
Go to symbol Ctrl + Alt + Shift + N 查找方法
Go to type declaration Ctrl + Shift + B 跳转到类型的定义处
Show bookmarks Shift + F11 显示所有的标签
Hide active or last active window Shift + Esc 隐藏当前（或最后活动的）工具窗口，并切换到编辑窗口
Go to previous / next method Alt + Up/Down 跳转上一个/下一个方法
Recent files popup Ctrl + E 弹出窗口列举出最近访问的文件
Navigate to last edit location Ctrl + Shift + Backspace 导航到最近编辑的位置（可以连续操作）
Move to code block end / start Ctrl + ]/[ 移动到代码块结束处/开始处，并高亮{}
Go to file Ctrl + Shift + N 查找文件
Go to numbered bookmark Ctrl + #[0-9] 跳转到指定标记的书签，配合Ctrl + F11使用
Go back to previous tool window F12 切换到最近使用的工具栏窗口

Live Templates

动作 快捷键 说明
Surround with Live Template Ctrl + Alt + J 选择代码块，添加到模板代码块中去
Insert Live Template Ctrl + J 插入模板代码

Usage Search

动作 快捷键 说明
Show usages Ctrl + Alt + F7 弹出列表窗口，显示出指定的类/方法被使用情况
Highlight usages in file Ctrl + Shift + F7 查找类/方法/变量使用情况，只找当前文件。类似于Ctrl + F效果
Find usages / Find usages in file Alt + F7 / Ctrl + F7 查找类/方法/变量使用情况。Ctrl + F7只找当前文件

Debugging

动作 快捷键 说明
Step over / into F8 / F7 逐行执行/进入方法内部
View breakpoints Ctrl + Shift + F8 查看所有断点信息
Evaluate expression Alt + F8 弹出窗口，输入验证表达式
Smart step into / Step out Shift + F7 / Shift + F8 智能进入方法内部/跳出方法
Resume program F9 跳到下一个断点，或者恢复运行（最后一个断点情况下）
Run to cursor Alt + F9 运行到光标处所在的行
Toggle breakpoint Ctrl + F8 当前行添加/删除断点

Search / Replace

动作 快捷键 说明
Find next / previous F3 / Shift + F3 查找下一个，Ctrl + F/R之后使用 /查找上一个，Ctrl + F/R之后使用
Replace Ctrl + R 查找替换文本，支持多行查找/只在代码中查找/只在注释中查找/正则表达式
Find in path Ctrl + Shift + F 在指定路径/整个项目中查找文本，支持文件过滤
Unselect occurrence Alt + Shift + J 取消选中
Find Ctrl + F 查找文本，支持多行查找/只在代码中查找/只在注释中查找/正则表达式
Search everywhere Double Shift 查找源代码中的任何条目，类似于Ctrl + Shift + N
Select all occurrences Ctrl + Alt + Shift + J 选中全部
Replace in path Ctrl + Shift + R 在指定路径/整个项目中查找替换文本，支持文件过滤
Select next occurrence Alt + J 选中下一个

Editing

动作 快捷键 说明
Generate code… Alt + Insert 生成代码，如：getter/setter/构造方法/…
Implement methods Ctrl + I 弹出窗口选择要Implement的方法
Show intention actions and quick-fixes Alt + Enter 显示意图采取行动和快速修复错误，可以自动导入包
Comment / uncomment with block comment Ctrl + Shift + / 注释/取消注释代码块：/…/
Paste from recent buffers… Ctrl + Shift + V 弹出窗口列举出最近使用的剪贴板内容，选择性插入
Smart line join Ctrl + Shift + J 合并选择的行为一行
Auto-indent line(s) Ctrl + Alt + I 自动缩进行
Surround with…  Ctrl + Alt + T 选择代码块，添加到if/try/for/…代码块中去
Extend selection Ctrl + W 选择连续增加的代码块（连续操作）
Indent / unindent selected lines Tab / Shift + Tab 缩进/取消缩进行
Close active editor tab Ctrl + F4 关闭活动的编辑器选项卡
Override methods Ctrl + O 弹出窗口选择要Override/Implement的方法
Parameter info (within method call arguments) Ctrl + P 显示方法的参数
Comment / uncomment with line comment Ctrl + / 注释/取消注释代码行：//…
Delete to word end Ctrl + Delete 从光标处删除到单词末尾
Shrink selection Ctrl + Shift + W Ctrl + W的相反操作，减少选择的代码块（连续操作）
Delete line at caret Ctrl + Y 删除当前行或者选择的行
Duplicate current line Ctrl + D 复制当前行到下一行（未选择行时）；复制选择的代码块到选择的末尾位置
Brief Info Ctrl + mouse 显示类、方法、变量概要信息
Quick documentation lookup Ctrl + Q 显示注释文档
Copy current line to clipboard Ctrl + C , Ctrl + Insert 复制当前行或者选择的代码块，并复制到剪切板
Expand all Ctrl +  Shift + NumPad+ 展开所有的代码
Toggle case for word at caret or selected block Ctrl + Shift + U 大小写转换
Collapse all Ctrl + Shift + NumPad- 折叠所有的代码
Paste from clipboard Ctrl + V , Shift + Insert 从剪切板粘帖
Select till code block end / start Ctrl + Shift + ] / [ 从光标处选择代码块到结束处/开始处
Optimize imports Ctrl + Alt + O 优化导入的类和包
Smart code completion Alt + Shift + / 自动补全代码，或者列举出相应方法和变量集合
Expand / collapse code block Ctrl + NumPad+ / - 展开/折叠代码块
Start new line Shift + Enter 向下插入新行
Delete to word start Ctrl + Backspace 从光标处删除到单词的开始
Cut current line to clipboard Ctrl + X , Shift + Delete 删除当前行或者选择的代码块，并复制到剪切板
Smart line split Ctrl + Enter 拆分行
External Doc Shift + F1 在浏览器中打开光标所在的类或者方法文档
Basic code completion Alt + / 自动补全代码，可用于类、方法、变量
Show descriptions of error at caret Ctrl + F1 显示错误或者警告信息
Complete statement Ctrl + Shift + Enter 补全当前行，如：在行末尾加;完成代码；if/else代码中加上{}完成代码块
Reformat code Ctrl + Alt + L 格式化代码
Context info Alt + Q 显示上下文信息，如方法/类声明，光标和声明处必须不在同一屏幕处，否则不会提示

General

动作 快捷键 说明
Toggle maximizing editor Ctrl + Shift + F12 切换最大化编辑器
Inspect current file with current profile Alt + Shift + I 检查当前文件与当前的配置文件
Open corresponding tool window Alt + #[0-9] 打开相应的工具窗口
Save all Ctrl + S 保存所有的文件
Synchronize Ctrl + Alt + Y 同步文件到磁盘
Quick switch current scheme Ctrl + BackQuote (̀ ) 快速切换配置信息，如：皮肤/快捷键/格式化代码风格等
Find Action Ctrl + Shift + A 查找菜单功能/快捷键设置/配置等信息
Open Settings dialog Ctrl + Alt + S 打开设置对话框
Open Project Structure dialog Ctrl + Alt + Shift + S 打开项目结构设置对话框

Refactoring

动作 快捷键 说明
Change Signature Ctrl + F6 更改签名
Copy F5 复制当前类或者文件
Move F6 移动类/方法/文件
Extract Constant Ctrl + Alt + C 提取选中的代码块生成一个新的常量
Safe Delete Alt + Delete 安全方式删除
Refactor this Ctrl + Alt + Shift + T 打开Refactor窗口
Inline Ctrl + Alt + N 将方法/类重构为内联方法/匿名类
Extract Parameter Ctrl + Alt + P 提取选中的代码块生成一个新的参数
Extract Variable Ctrl + Alt + V 提取选中的代码块生成一个新的变量
Extract Field Ctrl + Alt + F 提取选中的代码块生成一个新的属性
Rename Shift + F6 重命名方法/类/文件
Extract Method Ctrl + Alt + M 提取选中的代码块生成一个新的方法

Compile and Run

动作 快捷键 说明
Run / Debug Shift + F10 / F9 运行/调试
Make project Ctrl + F9 编译项目
Run context configuration from editor Ctrl + Shift + F10 从编辑器中的上下文配置运行，如：jUnit的test方法，ant中…
Compile selected file, package or module Ctrl + Shift + F9 编译选中的文件/包/模块
Select configuration and run / debug Alt + Shift + F10/F9 选择指定配置/文件运行/调试
